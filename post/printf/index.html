your own printf

library to build printf

printf using several function defined inside stdarg.h to build a string on stack, interleaving the string format with your data.

__printf(const char *format, ...)
{
  va_list arg;
  int done;
  va_start(arg, format);
  done = vfprintf(stdout, format, arg);
  va_end(list);
  return done;
}
va_list

the object will hold the list of additional arguments of the function. it will be empty until you initialize it with va_start.

va_start(typename format, …)

va_start initializes an object of this type in such a way that subsequent calls to va_arg sequentially retrieve the additional arguments passed to the function.

the argument before the variadic arguments starts will function at flag to tell va_start after the argument, the argument list start. it can be a format string as for printf, or if the arguments are all the same, use count as the argument to indicate size of the list.

va_arg

this function require type of the indexed argument to be specified.

va_end

after done with va_list object, you need call va_end to pair with va_start.
